# 奶茶店小料搭配评分预测器

## 项目介绍
一个基于 Vue 3 + Java 的奶茶店小料搭配评分预测系统，通过历史数据预测搭配的受欢迎程度。

## 技术栈
- **前端**: Vue 3 + Vite + Axios
- **后端**: Spring Boot 2.7.x + MyBatis Plus
- **数据库**: MySQL
- **缓存**: Redis
- **构建工具**: Maven (后端) / npm (前端)

## 项目结构
```
milk-tea-predictor/
├── backend/          # Java 后端项目
├── frontend/         # Vue 3 前端项目
└── database/         # 数据库脚本
```

## 功能特性
1. **茶底选择**: 红茶、绿茶、乌龙
2. **小料选择**: 珍珠、椰果、仙草、布丁、爆珠（可多选）
3. **评分预测**: 基于规则表的简单预测算法
4. **预测记录**: 每次预测都会记录到数据库
5. **热门搭配**: Redis 缓存热门搭配推荐
6. **类似推荐**: 基于当前选择推荐类似搭配
