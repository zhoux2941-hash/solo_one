# 幼儿园手工区材料消耗趋势仪表盘

## 项目概述
- **后端**: Java + Spring Boot 3.2 + JPA + MySQL
- **前端**: Vue 3 + Vite + ECharts
- **端口**: 后端 8080，前端 3000

## 启动步骤

### 1. 准备数据库
确保 MySQL 服务已启动，并创建数据库：

```sql
CREATE DATABASE kindergarten_material DEFAULT CHARACTER SET utf8mb4;
```

或者直接执行后端的 `schema.sql` 脚本。

### 2. 配置数据库连接
修改 `backend/src/main/resources/application.yml` 中的数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/kindergarten_material?useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
```

### 3. 启动后端服务

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

后端服务将在 http://localhost:8080 启动。

### 4. 安装前端依赖并启动

```bash
cd frontend
npm install
npm run dev
```

前端服务将在 http://localhost:3000 启动。

### 5. 访问应用
打开浏览器访问 http://localhost:3000

## API 接口

### 获取趋势数据
- **URL**: `GET /api/material/trend`
- **功能**: 返回过去 7 天的材料消耗趋势数据
- **响应示例**:
```json
{
  "dates": ["2026-05-03", "2026-05-04", ...],
  "materials": [
    {
      "name": "彩纸",
      "unit": "克",
      "data": [25.5, 32.1, ...]
    }
  ]
}
```

### 获取占比数据
- **URL**: `GET /api/material/share`
- **功能**: 返回各材料总消耗占比
- **响应示例**:
```json
{
  "total": 450.5,
  "items": [
    {
      "name": "彩纸",
      "unit": "克",
      "amount": 180.5,
      "percentage": 40.06
    }
  ]
}
```

## 材料类型
| 材料名称 | 单位 | 每日 Mock 数据范围 |
|---------|------|-------------------|
| 彩纸 | 克 | 20 ~ 50g |
| 胶水 | 毫升 | 5 ~ 15ml |
| 亮片 | 克 | 3 ~ 10g |
| 毛根 | 个 | 5 ~ 20个 |
