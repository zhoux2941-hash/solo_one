# 便利店自助收银与会员积分系统

基于原生 HTML/JS 前端 + Java Spring Boot 后端的便利店自助收银机模拟系统。

## 功能特性

### 商品管理
- 📱 扫码添加商品（模拟输入商品条码）
- 前端展示商品名称、单价、数量
- 快捷商品点击添加
- 支持修改商品数量
- 支持删除商品
- 实时计算商品总价

### 会员积分系统
- 👤 会员手机号验证
- 会员享受 95 折优惠
- 消费累计积分（1元 = 1积分）
- 实时显示会员信息和积分

### 支付与小票
- 💳 确认支付
- 生成电子小票
- 打印小票（模拟）
- 支持新订单

## 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.6+

### 启动步骤

1. **进入项目目录**
   ```bash
   cd e:\trae-project\0508-194
   ```

2. **编译项目**
   ```bash
   mvn clean compile
   ```

3. **启动应用**
   ```bash
   mvn spring-boot:run
   ```

4. **访问系统**
   打开浏览器访问：http://localhost:8080

## API 接口

### 商品接口
- `GET /api/product/{barcode}` - 根据条码查询商品
- `GET /api/products` - 获取所有商品列表

### 会员接口
- `GET /api/member/{phone}` - 根据手机号查询会员
- `POST /api/member/register` - 注册新会员

### 收银接口
- `POST /api/checkout` - 结账支付

## 使用说明

1. **添加商品**
   - 在输入框中输入商品条码，按回车或点击"添加商品"
   - 或直接点击下方快捷商品卡片

2. **管理购物车**
   - 点击 +/- 按钮修改商品数量
   - 点击"删除"按钮移除商品
   - 数量为0时自动移除

3. **会员优惠**
   - 输入会员手机号，点击"验证会员"
   - 验证成功后自动享受95折并显示会员信息

4. **结算支付**
   - 确认商品无误后点击"确认支付"
   - 系统生成电子小票
   - 可选择"打印小票"或开始"新订单"

## 注意事项

- 本系统使用内存存储数据，重启后数据会重置
- 预置了测试商品和会员，可直接使用测试
- 打印功能为模拟，实际打印需要对接打印机驱动
